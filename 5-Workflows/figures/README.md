# Workflow Figures

Concept illustrations embedded in the [workflow tutorial](../README.md) to picture each pattern before the code appears.

Unlike the [`diagrams/`](../diagrams/README.md) folder — which holds plots auto-generated from *this repo's* graphs — these are **static teaching figures** that illustrate the patterns in the abstract. They are not regenerated by any script here; treat them as fixed reference art.

**Attribution:** these figures are adapted from the workflow-types deck cited below and are included purely as third-party reference material for the concepts. The tutorial's own explanations, the Mermaid diagrams, and the code are written originally from this repo's examples — the figures illustrate ideas, they are not the source of the lessons.

Source: [Workflow types deck](https://docs.google.com/presentation/d/1SuUtO_OCcglql3DRU5KnbKtxxHokxPhX/edit?slide=id.p2#slide=id.p2)
