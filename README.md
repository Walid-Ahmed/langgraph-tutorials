# LangGraph Tutorials

Simple LangGraph examples for learning graph basics, reducers, message state, and conditional edges.

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Run Examples

```bash
python "1-Langgraph basics/00_simple_graph.py"
python "Conditional Edges/05_conditional_edges.py"
```

For LLM examples, create a local `.env` file with your OpenAI API key:

```bash
OPENAI_API_KEY=your_api_key_here
```
